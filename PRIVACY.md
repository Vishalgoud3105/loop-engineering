# Privacy Policy — loop-engineering

`loop-engineering` is a Claude Code plugin consisting entirely of markdown
skill definitions. It does not collect, store, transmit, or process any
user data, telemetry, or analytics of any kind.

It has no backend service, no network calls of its own, and no third-party
integrations beyond invoking other Claude Code skills already present in
your session (`ponytail`, `code-review`, `debugging-code`, `verify`,
`security-review`) — each of those operates under its own terms, not this
plugin's.

Any code, files, or project content this plugin reads or writes stays
local to your machine and your Claude Code session, the same as any other
Claude Code skill or command you invoke directly.

Questions: open an issue at
[github.com/Vishalgoud3105/loop-engineering](https://github.com/Vishalgoud3105/loop-engineering/issues).
